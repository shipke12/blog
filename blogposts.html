What is Responsive Web Design?

w3schools.com/whatis/whatis_responsive.asp
Sarah Hipke
2/16/2020

Responsive web design is a design meant for automatically resizing your website depending on the device and browser it is being accessed in. W3Schools explained a method of using media queries. These queries act as breakpoints in the page for when the site needs to resize. They work in real time as the user makes the window smaller or larger.


<style>
.left, .right {
float: left;
width: 20%; /* The width is 20%, by default */
}

.main {
float: left;
width: 60%; /* The width is 60%, by default */
}

/* Use Media Query to add a breakpoint at 800px: */
@media screen and (max-width:800px) {
.left , .main, .right {width:100%;}
}
</style>


Images are often affected with resizing windows as well, and there are a few different ways of having them adjust. An easy way to ensure the image resizes perfectly every time in any browser is to set the width to a percentage. It will automatically adjust the picture and height keeping the aspect ratio in tact. Otherwise, using the picture element let's you change the picture to another at certain window sizes. The article made sure to mention Bootstrap as a tool used to make responsive webpages as it combines HTML, CSS and jQuery all into one.
